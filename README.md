# Wallpaper
Create random polygon wallpapers

Install Required Modules:
```bash
pip install pillow

pip install colorama
pip install termcolor
```

```bash
mkdir Wallie
cd Wallie
git clone https://github.com/Atikul10152002/Wallpaper.git
cd Wallpaper
python wallpaper.py
ls walls
```
